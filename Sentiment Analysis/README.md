# Movie sentiment analysis

Dataset link: https://www.kaggle.com/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

**Methodology:**
1. Import packages and dataset
2. Convert dataset into np array and preprocess text 
3. Preprocessing involves removal of non-alphabets, removal of stop words and stemming.
4. Tokenize dataset using Tfidfvectorizer  
5. Train and Test splitting 
6. Model building (Multinomial Navie Bayes classifier is used) and prediction
7. Accuracy achieved is 85%.
